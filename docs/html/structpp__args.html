Detailed Description

Pretty Print.

A struct with virtual function pp that can be added to any class to pretty print it's representation.

Example usage:

---MyClass.h---
class MyClass : public PP
{
public:
void pp(pp_args args = pp_args()) override;
}
---MyClass.cpp---
void MyClass::pp(pp_args args)
{
cout << args.ident << name << endl;
}
