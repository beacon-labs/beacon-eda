{\bfseries{Doxygen Awesome}} is a custom {\bfseries{CSS theme for Doxygen HTML-\/documentation}} with lots of customization parameters.\hypertarget{md_doxygen_awesome_css_README_autotoc_md30}{}\doxysection{Motivation}\label{md_doxygen_awesome_css_README_autotoc_md30}
I really like how the Doxygen HTML-\/documentation is structured! But IMHO it looks a bit outdated.

This theme is an attempt to update the visuals of Doxygen without changing its overall layout too much.\hypertarget{md_doxygen_awesome_css_README_autotoc_md31}{}\doxysection{Features}\label{md_doxygen_awesome_css_README_autotoc_md31}

To use the theme in your documentation, copy the required CSS and JS files from this repository into your project or add the repository as submodule and check out the latest release\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git submodule add https://github.com/jothepro/doxygen-\/awesome-\/css.git}
\DoxyCodeLine{cd doxygen-\/awesome-\/css}
\DoxyCodeLine{git checkout v2.1.0}

\end{DoxyCode}


All theme files are located in the root of this repository and start with the prefix {\ttfamily doxygen-\/awesome-\/}. You may not need all of them. Follow the install instructions to figure out what files are required for your setup.\hypertarget{md_doxygen_awesome_css_README_autotoc_md34}{}\doxysubsection{Choosing a layout}\label{md_doxygen_awesome_css_README_autotoc_md34}
There is two layout options. Choose one of them and configure Doxygen accordingly\+:





\hypertarget{md_doxygen_awesome_css_README_autotoc_md35}{}\doxysubsubsection{Base Theme (1)}\label{md_doxygen_awesome_css_README_autotoc_md35}
Comes with the typical Doxygen titlebar. Optionally the treeview in the sidebar can be enabled.

Required files\+: {\ttfamily doxygen-\/awesome.\+css}

Required {\ttfamily Doxyfile} configuration\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{GENERATE\_TREEVIEW      = YES \# optional. Also works without treeview}
\DoxyCodeLine{DISABLE\_INDEX = NO}
\DoxyCodeLine{FULL\_SIDEBAR = NO}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET  = doxygen-\/awesome-\/css/doxygen-\/awesome.css}

\end{DoxyCode}
\hypertarget{md_doxygen_awesome_css_README_autotoc_md36}{}\doxysubsubsection{Sidebar-\/\+Only Theme (2)}\label{md_doxygen_awesome_css_README_autotoc_md36}
Hides the top titlebar to give more space to the content. The treeview must be enabled in order for this theme to work.

Required files\+: {\ttfamily doxygen-\/awesome.\+css}, {\ttfamily doxygen-\/awesome-\/sidebar-\/only.\+css}

Required {\ttfamily Doxyfile} configuration\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{GENERATE\_TREEVIEW      = YES \# required!}
\DoxyCodeLine{DISABLE\_INDEX          = NO}
\DoxyCodeLine{FULL\_SIDEBAR           = NO}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET  = doxygen-\/awesome-\/css/doxygen-\/awesome.css \(\backslash\)}
\DoxyCodeLine{                         doxygen-\/awesome-\/css/doxygen-\/awesome-\/sidebar-\/only.css}

\end{DoxyCode}
